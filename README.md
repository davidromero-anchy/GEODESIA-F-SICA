# Geodesia Física - Potencial de Figuras Simples

Aplicación interactiva desarrollada en Streamlit para visualizar cuatro demostraciones matemáticas de geodesia física:

- Módulo 1: Gravedad interior y exterior
- Módulo 2: Potencial del cascarón esférico
- Módulo 3: Potencial de la esfera homogénea
- Módulo 4: Anomalía gravimétrica del cilindro vertical

## Ejecutar localmente

```bash
pip install -r requirements.txt
streamlit run app.py
